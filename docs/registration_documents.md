# 法人設立登記に必要な書類一覧

## 合同会社（LLC）

### アプリで自動生成できる書類
1. **定款** - 既存機能で対応済み
2. **本店所在場所の決定を証する書面** - 社員全員の同意書形式
3. **資本金の額の決定を証する書面** - 代表社員の証明書形式
4. **業務執行社員・代表社員の選任を証する書面** - 社員総会議事録形式
5. **業務執行社員・代表社員の就任承諾書** - 各社員分
6. **払込みがあったことを証する書面（払込証明書）** - 通帳コピー貼付台紙付き
7. **合同会社設立登記申請書** - 法務局提出用
8. **印鑑届出書** - 代表社員の印鑑届

### ユーザーが自分で用意する書類（案内のみ）
- 代表社員の印鑑登録証明書（市区町村で取得）
- 登録免許税納付用台紙（収入印紙6万円）
- 印鑑カード交付申請書（法務局で取得）

## 株式会社

### アプリで自動生成できる書類
1. **定款** - 既存機能で対応済み
2. **株式会社設立登記申請書**
3. **発起人の決定書**（本店所在場所・設立時役員の決定）
4. **設立時取締役の就任承諾書**
5. **設立時代表取締役の就任承諾書**
6. **払込証明書**（資本金の払込みを証する書面）
7. **資本金の額の計上に関する証明書**

### ユーザーが自分で用意する書類（案内のみ）
- 設立時取締役・代表取締役の印鑑証明書
- 登録免許税納付用台紙（収入印紙15万円）
- 定款認証費用（公証役場）

---

## 一般社団法人

### アプリで自動生成できる書類
1. **定款** - 既存機能で対応済み
2. **一般社団法人設立登記申請書**
3. **設立時社員の決議書**
4. **設立時理事の就任承諾書**
5. **設立時代表理事の就任承諾書**

### ユーザーが自分で用意する書類（案内のみ）
- 設立時理事・代表理事の印鑑証明書
- 登録免許税納付用台紙（収入印紙6万円）
- 定款認証費用（公証役場）
